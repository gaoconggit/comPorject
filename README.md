 * vuex             --- 数据存储
 * axios            --- ajax请求库
 * localStorage     --- 本地存储
 * vux              --- UI库

# 安装依赖
1. node.js
2. `npm install` or `yarn install`
3. 运行项目
   `npm run dev` or `yarn run dev`
4. 打包项目
   `npm run build` or `yarn run build`
